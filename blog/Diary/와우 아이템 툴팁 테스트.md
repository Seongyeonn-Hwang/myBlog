# 와우 아이템 툴팁 테스트

마크다운 포스트에서 와우 아이템 이름에 마우스를 올리면 아이템 정보를 볼 수 있어요.

## 사용 방법

`[wow:아이템ID|표시이름]` 형식으로 작성합니다.
품질(색상) 지정 시: `[wow:아이템ID|표시이름|q품질]` (q0~q7)

| 품질 코드 | 품질 |
|---------|------|
| q0 | 형편없는 (회색) |
| q1 | 일반 (흰색) |
| q2 | 고급 (녹색) |
| q3 | 희귀 (파란색, 기본값) |
| q4 | 영웅 (보라색) |
| q5 | 전설 (주황색) |
| q6 | 유물 (황금색) |

## 예시 아이템들

전설 클래식 아이템: [wow:19019|천둥분노, 바람추격자의 축복받은 검|q5]

전설 클래식 아이템: [wow:17182|술루마르, 정복자의 눈|q5]

영웅 아이템 예시: [wow:34428|검은 날의 검|q4]

희귀 아이템 예시 (기본값): [wow:1168|구리 단검]

고급 아이템 예시: [wow:25|장화|q2]

아이템 이름 위에 마우스를 올려보세요!
