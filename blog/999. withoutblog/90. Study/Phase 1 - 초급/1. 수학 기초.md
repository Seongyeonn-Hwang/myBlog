### 📚 학습 순서

## 2️⃣ 미적분 (Calculus)

### 📚 학습 순서

#### Week 3: 미분의 기초

**개념**
- **도함수 (Derivative)**: 함수의 변화율
- **연쇄법칙 (Chain Rule)**: 합성함수의 미분
- **편미분 (Partial Derivative)**: 여러 변수 중 하나로만 미분

**중요 공식**
```
f(x) = x²의 도함수 = 2x
f(x) = sin(x)의 도함수 = cos(x)

다변수함수 z = f(x, y):
∂z/∂x = y를 상수로 취급하고 x로만 미분
∂z/∂y = x를 상수로 취급하고 y로만 미분
```

**그래디언트 (Gradient)**
```python
# 그래디언트: 가장 가파른 방향을 가리킨다
# ∇f = [∂f/∂x, ∂f/∂y, ...]

# 예: f(x, y) = x² + y²
# ∇f = [2x, 2y]  --> (x, y)에서 가장 가파른 방향

# 이것이 경사하강법(Gradient Descent)의 기초!
```

#### Week 4: 경사하강법

**개념**
- **경사하강법 (Gradient Descent)**: 최솟값을 찾는 알고리즘
- 그래디언트의 반대 방향으로 이동하면서 최솟값 찾기

**알고리즘**
```python
def gradient_descent(f, grad_f, x_init, learning_rate=0.01, iterations=100):
    """
    f: 목적함수
    grad_f: 그래디언트 함수
    x_init: 초기값
    """
    x = x_init
    history = [x]

    for _ in range(iterations):
        gradient = grad_f(x)
        x = x - learning_rate * gradient  # 그래디언트 반대 방향으로 이동
        history.append(x)

    return x, history

# 예: f(x) = x²의 최솟값 찾기
f = lambda x: x**2
grad_f = lambda x: 2*x
optimal_x, history = gradient_descent(f, grad_f, x_init=5.0)
```

**AI에서의 중요성**
- 신경망 훈련의 핵심 알고리즘
- 손실함수를 최소화하는 가중치 찾기

## 3️⃣ 확률과 통계 (Probability & Statistics)

### 📚 학습 순서

#### Week 5: 확률의 기초

**개념**
- **확률 (Probability)**: 사건이 일어날 가능성
- **확률분포**: 확률의 분포
- **기댓값과 분산**: 평균과 퍼짐의 정도

**기본 공식**
```
P(A) = 유리한 경우의 수 / 전체 경우의 수

조건부 확률:
P(A|B) = P(A ∩ B) / P(B)  (B가 주어졌을 때 A의 확률)

베이즈 정리:
P(A|B) = P(B|A) × P(A) / P(B)
```

**확률분포**
- **정규분포 (Normal Distribution)**: 종 모양의 분포 (가장 중요)
- **이항분포 (Binomial Distribution)**: 성공/실패 확률
- **포아송분포 (Poisson Distribution)**: 시간당 사건 발생 횟수

#### Week 6: 통계 기초

**개념**
- **표본과 모집단**: 표본으로 모집단 추론
- **중심극한정리**: 표본평균은 정규분포를 따른다
- **신뢰도와 신뢰구간**: 추정의 확실성

**중요 개념**
```python
import scipy.stats as stats

# 정규분포에서 표본 추출
samples = np.random.normal(loc=100, scale=15, size=1000)

# 표본 통계
mean = np.mean(samples)  # 평균
std = np.std(samples)    # 표준편차
confidence_interval = stats.t.interval(0.95, len(samples)-1,
                                       loc=mean, scale=std/np.sqrt(len(samples)))
```

**가설검정**
- H0(영가설): 귀무가설
- Ha(대립가설): 연구가설
- p-value: 영가설이 참일 확률

### 실습 프로젝트

```python
# 프로젝트: 주사위 던지기 분석
import numpy as np
from scipy import stats

# 주사위 1000번 던지기
rolls = np.random.randint(1, 7, size=1000)

# 각 숫자 빈도
for i in range(1, 7):
    count = np.sum(rolls == i)
    print(f"{i}: {count/1000:.3f} (예상: {1/6:.3f})")

# 가설검정: 주사위가 공정한가?
chi2, p_value = stats.chisquare(np.bincount(rolls)[1:])
print(f"p-value: {p_value:.4f}")
if p_value > 0.05:
    print("주사위는 공정해 보입니다 (유의수준 5%)")
```
